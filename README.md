# AI Service - Audit Trail and Activity Log Viewer

## Overview

This AI service is built using Flask and provides AI-powered audit log analysis features for the Audit Trail and Activity Log Viewer project.

The service supports:
- Audit activity description generation
- Security recommendations
- Audit report generation
- Health monitoring

---

## Tech Stack

- Python 3.11
- Flask
- Flask-CORS
- REST API
- Docker
